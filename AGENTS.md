# Hl2 map ratings website

This is a fun static website used to systematically catalogue and rate user made half life 2 maps.

# Agent instructions

- You can store temporary files (plans, architecture notes etc) in the agent-temp folder. This folder is gitignored, so you can do whatever you want in here.
- When you implement a new feature or an improvement for the user, write VERY concise summary of why you did what you did, and store it in the agent-temp/summaries folder. The user is not just interested in vibe coding, they want to learn from you. In these files have a brief description of the problem, what you tried, what was the solution and why did you choose that solution. The user is interested in the key learning points of the changes you made.
